Model Assumptions & Scope

  • All costs are annualized for the production volume specified
  • Fill volume with overfill = nominal fill × (1 + overfill%). WFI system provides sterile water for reconstitution if needed
  • In-process yield: vials rejected at IPC checkpoints (weight, cosmetic, particulate). Batch yield applied at vial level
  • Release rate: entire batches that fail final QC (sterility, potency, endotoxin). Applied at batch level
  • Blended yield loss = 1 − (batchYield% × releaseRate%). Both levels modeled simultaneously
  • Fill runtime = vialsPerBatch ÷ (ratedSpeed × OEE%). OEE captures availability, performance, and minor stops
  • Changeover includes line clearance, CIP/SIP, depyrogenation tunnel, component staging, setup qualification
  • Line utilization % applies to: Facility CAPEX, Line CAPEX, Facility OPEX (maintenance + regulatory), Line OPEX (utilities + maintenance). Does NOT reduce QC-per-batch, direct/indirect labor, materials, or packaging
  • CAPEX annualized via straight-line depreciation. No residual value assumed
  • Direct labor: FTE × prodDays × shifts × hrs/shift × rate × lineUtil%. Operators follow the shared facility schedule
  • Indirect labor: FTE × 2,080 hr/yr × rate × lineUtil%. Support staff partially shared with other products
  • Yield Loss = rejected vials × (API + component cost/vial). Excipients and consumables are sunk per batch → in Materials
  • Materials & Consumables covers released product API + components, all excipients, all batch consumables
  • Packaging & Labeling applied only to released (saleable) vials
  • 3rd-party CAPEX toggle: when OFF, Facility CAPEX and Line CAPEX = $0. All OPEX still applies
  • Profit margin applied as: total cost ÷ (1 − margin%). Margin shown as a cost line for transparency
  • Lyophilization toggle: when ON, lyo CAPEX + maintenance + consumables are modeled as a separate cost line; freeze-dry cycle time + load/unload is added to batch cycle time, reducing annual throughput significantly. Exclude lyophilizer from Line CAPEX when using this toggle
  • Isolator/RABS and auto-inspection system should be included in Line CAPEX if applicable
  • Cold-chain packaging, distribution, and post-market surveillance not included
